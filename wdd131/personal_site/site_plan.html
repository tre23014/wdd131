Overview

Purpose

It is going to be a quiz website so we are gathering basic information.

Audience

For those who would like change not only their health for the better, but their habits, mindset, home, and overall way of life!

Dynamic elements

I am thinking of using DOM Interaction by using JavaScript to display the options, Event Listeners for changing the options, Objects to hold the options/questions so they are all in one place, Using calculation to see how many they clicked vs avaiable, and using ES modules for organization.
